Autarky search results

Selected benchmark for the evaluation: DQBF Track QBFEVAL'18.
Total number of instances: 334.
Instances with no non-trivial autarkies: 330.
Instances with a non-trivial autarkies: 4.

With improved SAT-encodings and running standard SAT-solvers, we where able to compute the normalforms for all 334 instances in the DQBF track of QBFEVALâ€™18. All normalforms have been computed within a total time of 9000s. (We remark that finding an autarky is quick, and most of the time is spent on proving unsatisfiability.)

The evaluation contained 334 instances, where 330 do not contain any non-trivial A1- or E1-autarky. The four remaining instances with non-trivial autarkies are

  1. bloem_eq1.dqdimacs
  2. tentrup17_ltl2dba_theta_environment_1.dqdimacs
  3. bloem_ex1.dqdimacs
  4. bloem_ex2.dqdimacs

We use n(F) to denote total number of variables, c(F) for number of clauses, a(F) for total number of universal variables and
e(F) for total number of existential variables in the formula F. The overview of the results of the autarky reduction is presented in the following table.
